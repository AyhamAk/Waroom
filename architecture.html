Integrated Tools
✅
JS Syntax Validator
tools/validator.js · vm.Script
Validates every JavaScript file with Node's vm.Script before writing to disk. On failure, makes a focused LLM fix-call with the broken code + error message. Retries with the fixed code and skips if still invalid.
🔗
CDN Resolver
tools/cdnResolver.js · cdnjs API
Scans every HTML file for <script src> and <link href> tags. Replaces guessed or outdated library URLs with verified cdnjs.cloudflare.com URLs. Prevents 404 blank screens from bad CDN guesses.
🔍
Web Search
tools/search.js · Serper API
CEO and Lead Engineer optionally search the web before making decisions. Gives agents access to current market data, documentation, and best practices — beyond their training cutoff.
⚡
sucrase Transpiler
npm sucrase · TypeScript → JS
Real TypeScript-to-JavaScript transpiler. Applied to every .js file and inline <script> blocks before writing. Eliminates type annotations, interfaces, generics, and access modifiers that would crash the browser.
Auto-Systems (no human trigger)
🎨
CSS Specialist
Runs after cycle 1 builder · 10K tokens
A dedicated LLM call (smart model) that reads the actual HTML on disk, extracts every id="" and class="", then writes a COMPLETE stylesheet targeting exactly those selectors. Replaces whatever CSS the builder wrote. Guarantees dark theme, responsive layout, hover states, animations.
🔍
CSS Coverage Patcher
Runs every cycle · triggers at <60% coverage
After every cycle, counts how many HTML selectors have matching CSS rules. If coverage drops below 60%, makes a patch call listing only the uncovered selectors. Appends the missing rules to style.css. Prevents selector drift across multi-cycle builds.
📋
File Ensurer
Runs every cycle · guaranteed files
After every builder run, checks that index.html, style.css, and app.js exist. If any are missing, auto-generates them using the smart model with full context of everything already built. No cycle can end with missing core files.
